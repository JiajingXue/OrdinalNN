Jiajing Xue xuejiajing@stu.xmu.edu.cn

# Usage

* function.py

  * OrdinalNN

    Function used to estimate the proposed *Ordinal Sparse Neural Networks for Modeling Gene- and Imaging-Environment Interactions* model.

  * DGP

    Function used to generate simulated data.

* demo.py

  One example for usage of *OrdinalNN* function in estimating the G-E and I-E interaction models.
